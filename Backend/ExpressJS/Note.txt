Express.js :
--Express.js is a light weight Node.js framework design to make the development of server side applications straight forward by providing inbuilt methods for routing, handling http request/response and intigrating middleware easily
--Express.js is a minimalist web application framework built on Node.js which is primarily use to simplify the process of creating web servers.
--Express.js is flexible and fast which enables developers to built both small and large applications.
--It is commonly used to create APIs, web application and handling various http methods.

Use of Express.js :
--Built Web App and API using routing and middleware
--Reduces boilerplate code and coding time compare to VANILA Node.js
--Organises functionality with middleware for tasks like authentication, login, and errorhandling
--It allows rapid development of scalable and maintainable web application

Differences between Express.js and Node.js
--Routing :(Node.js) -> In Node.js manual setup mode code require
           (Express.js) -> Clean and builtIn routing system

--JSON Parsing :(Node.js) -> manual or 3rd party
                (Express.js) -> simple by using Express.json()

--Middleware :(Node.js) -> needs 3rd party library
              (Express.js)-> BuiltIn and systematic middleware support

HOW to create an Express.js project ?
--To create an Express.js project firstly we need Node.js Installed with npm
--And then Install the Express.js package using npm
--After that we have to setup our application step by step

Steps to create an express project
step 1 -Initialize project 
Open terminal and run the command to initialize npm 
-> npm init -y

step 2 -Install express
Install express via npm
->npm install express

step 3 - Create server files
Create a file named index.js or server.js as mentioned in your package.json file.
The file should be present in root directory of the project.

step-4 -Import express module and initialize the express.
const express=require('express')
const app=express()

step-5 -Define a port and start/listen the server
const PORT=8000
app.listen(PORT,()=>{
     console.log("Server running")
})

Understanding request and response in Express.js :-
request: It represents the http request and contains data sent by the client including path parameters, query strings body data, headers and more. 
response : It represents http response that an express app sends back to the client, it provides methods to send data like 
     res.send(), res.json(), res.status()......etc 

app.get("/",(req,res)=>{
    res.send("This is an example of response")
    console.log(req.method)
})
